# JSON Tree Visualizer

### About the Project  
The **JSON Tree Visualizer** is a web application that takes JSON input and visualizes it as a clean, interactive hierarchical tree.  
It displays object and array relationships clearly in a left-to-right layout, helping users understand complex JSON data structures easily.
